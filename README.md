# nifty_cloud_spa
[ING]Nifty CloudのMobile Backendを利用したサンプルアプリケーション（iOS Sample Study: Swift）

Nifty CloudのMobile Backendのデータストアを利用した、買い物メモのサンプルアプリケーションのサンプルになります。
今回はデータに加えてカメラでの画像も一緒に追加・変更・削除を行うことができるだけのシンプルなものです。
